# First time setup

* install nodejs
* install Yarn
* install Docker

```bash
yarn install
```

# Starting

## Start Local MongoDB

```bash
./start-db.sh
```

In first terminal start build process in watch mode:

```bash
yarn watch
```

In second terminal start an application:

```bash
yarn start
```

# API Browser

Open http://localhost:4000 to see Graphql Playground
